Angular + OpenStreetmap

Start to use openstreetmap with your favorite framework

What

The library angular-osm provide set of services that let you use all well known OpenStreetMap APIs

  • OpenStreetMap API (to get&edit data)
  • Overpass API (to query data)
  • TagInfo to get information on tags
  • Oauth to let OSM contributors connect to your APP
  • Nominatim
  • OSRM

Why

Angular is a framework to build APPs, OSM is a huge GIS database, so lets build app to make it easy to contribute.

How

You are not force to use the all library you have multiple build, so you can choose what every you want for your app

  • osm-api.js to only use the XML official api
  • osm-oauth.js to just use the oauth
  • osm-overpass.js to just be able to query data
  • osm-full.js to use them all together
  • osm-nominatim to only use nominatim api
  • osm-osrm to only use osrm api

You can check the examples (form the menu) to learn how to use each of them.
